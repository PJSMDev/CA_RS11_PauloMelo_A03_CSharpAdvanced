Repositório - o que é

	Padrão de design que abstrai o acesso a uma coleção de objetos
	Centraliza a lógica para aceder, adicionar, remover e atualizar objetos



Repositório - vantagens

	Abstração de dados
		Esconder os detalhes de como os dados são trabalhados
	
	Testabilidade
		Permitir a substituição de implementações reais por mocks em testes unitários

	Organização
		Manter a lógica de acesso a dados separada da lógica de negócios
